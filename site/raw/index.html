💡 Witney

Don't wait to start with your javascript and typescript projects.

Read Getting started to learn how to start using Witney for your next projects.

Features

  • 📦 Bundles with microbundle
  • 🧹 Formatting with prettier
  • 👨🏼‍⚖️ Typescript support with ts-node and ts-jest
  • 📏 Tests and coverage with jest
  • 🧢 Markdown site generation with mallery
  • 🎰 Versioning with standard-version
  • 🧰 Packed with commonly used dev scripts
    • All written with 🧩 extensibility in mind
    • Scripts for Building, Testing, Coverage, Server starting
    • 🔖 Badges generator
    • 🧐 Dependency license check
    • 👽 Dependency gyp check
